正经的图千篇一律，不正经的叔万里挑一

September 20, 2017 in R, Visualization

![](https://guangchuangyu.github.io/blog_images/R/geofacet/Screenshot 2017-10-17 10.31.23.png)

这个题目可以放好多不正经的图，我在这个公众号里面已经写过太多，比如《邪恶的曲线》和它的3D版本，《3D版邪恶的曲线》，但这里不涉及太多，只用geofacet这个包为例。

在《按地理位置分面》展示了它正经的一面，用来将数据可视化按照地理位置分面，比如下面这图就展示了欧洲各国人均GDP的情况，分面按照各国的相对位置来。

![](http://guangchuangyu.github.io/blog_images/R/geofacet/Screenshot 2017-06-22 10.25.41.png)

Y叔不正经起来呢，用它来画一只复活节的火鸡《ggplot2分面之像素艺术》。

![](http://guangchuangyu.github.io/blog_images/R/geofacet/Screenshot 2017-06-26 19.44.30.png)

苹果手机坏了，配合Y叔写的emojifont，用emoji组一个苹果来纪念它，《你所没见过的苹果》：

Y叔还写了一个ggimage包，让图上嵌图，拼图变得格外容易，并且内置了许多图片集可以用，比如geom_phylopic可以用phylopic数据库的图，geom_emoji可以用twitter版本的emoji，geom_pokemon可以用pokemon等等，这些内置的图层可以让你随意用这些图片来画图。在《ggimage：ggplot2中愉快地使用图片》里已经有各种示范。用宝可梦画图是件特别有趣的事情，Y叔又示范了使用宝可梦拼出了名字缩写，《我的名字和我的姓氏》，用ggimage来拼出各种形状的照片墙，轻而易举：

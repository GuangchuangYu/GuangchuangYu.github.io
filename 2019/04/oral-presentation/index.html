那些我讲过的oral

April 25, 2019 in Presentation

2016潮博沙龙

《潮博沙龙》系受到潮博群里大家的邀请，应邀去普及一下生信以及介绍自己的一些经历。

2016中国R语言大会

中国R语言大会，我到目前为止，只参加过2016年那一次，也就是第9届，那也是首次有Bioconductor分会，并且还邀请了bioconductor的老大martin morgan，我当时也是受邀请去参会的，虽然会议在人民大学举行，虽然主会场和其它分会场都是讲中文，但bioconductor分会场是要求讲英文的。因为主讲人一半是歪果仁，一半是中国人，而3个中国人之中又有两个是美帝来的，只有我一土鳖。本来邀请人跟我提了一下ChIPseeker，暗示可以讲这个包，但我强行去讲ggtree，然后会后，很多听众问我clusterProfiler的问题，也是挺尬聊的。

2017年进化树可视化

2017年受邀请去美国自然历史博物馆讲ggtree，这是field museum内部邀请举行的一个内部的会议，由他们的NSF基金资助，我带着统统一起去，省掉了统统去参观博物馆三十多刀的门票。被邀请都是在进化树可视化上有重要作品的人，讨论的是进化树可视化未来的发展，有很多都是大佬，比如写Mesquite的David Maddison，他还有个双胞胎兄弟，Wayne Maddison，两兄弟是进化界的翘楚，NEXUS格式做为进化树的储存格式就是他们兄弟俩提出来的。

Maddison DR, Swofford DL, Maddison WP (1997). “NEXUS: An extensible file format for systematic information”. Systematic Biology. 46 (4): 590–621

当时我走在路上，看到David Maddison，还在讲“估计前面的老人家也是来这里参加这个会议的”，不认识大佬的我，真是太囧了。

还有写phytools的Liam J. Revell，他私下问我要不要去美国做博后，被我无情地拒绝了，我跟他说我想在中国发展。

我是被邀请参加的唯一一个博士生，他们都很惊讶我竟然还没毕业，也对ggtree是由我自己一个开发的，表示惊讶。

2017年环境微生物

环境微生物的会其实是找工作的，当时联系了中大环境学院，一切都挺OK，但最终在学校层面被拒了，卡我没有两年的博士后经历。当时讲的主要也是我博士期间做的东西。后来年底又申请了中大医学院，但他们院长竟然忘记提交我的申请，来年我询问说忘记了，叫我联系某某老师，我也是服了。

2018南医大找工作啊找工作

《Y叔要求职》一文发出来之后，南医大有不少人联系我，3月份我来到了南医大，做了一次报告，然后我就确定来南医大工作了。机遇就在转角处，本来我都没关注南医大，但有时候就是这样，有机会也有贵人。

2018南湖论坛

华中农的南湖论坛，其实也是找工作的，有国外的教授推荐我去，我说我没有去武汉工作的意向，教授对我说，谁说一定要去工作的？去推销一下自己的工作，认识些新朋友就挺好，于是我就去了，顺道带着在武汉读大学的统统去重走青春路。我本来报的不是信息学院，据说被强行选去信息学院讲。我当时讲的是treeio + ggtree。就在我到武汉的第二天，我收到了南医大的offer，而第三天，我收到了中大医学院面试的通知，然后我告知他们，我已经拿了offer，reject他们的面试。

2019南医大基础医学院年会

刚来就被拉出来溜溜，刚好讲MBE上的文章：《文章发表：两种方法在进化树上可视化数据》。
